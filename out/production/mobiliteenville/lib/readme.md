# External Libraries

This directory contains external libraries that are used in this project. 

- [json-20250517.jar](json-20250517.jar) : A library for working with JSON data in Java. More information can be found at [JSON-java GitHub repository](https://github.com/stleary/JSON-java)  
 - [opencsv-3.9.jar](opencsv-3.9.jar) : A library for reading and writing CSV files in Java. More information can be found at [OpenCSV SourceForge page](http://opencsv.sourceforge.net).
 - [JadeUPHF.jar](JadeUPHF.jar) : A library for integrating JADE-UPHF agents platform.  More information can be found at [JadeUPHF GitHub repository](https://emmanueladam.github.io/JadeUPHF/)
